The `apt-get` or `apt` command downloads packages easily.

# Usage

`apt-get <option>`
`apt <option>`

```shell
$ sudo apt update
$ sudo apt upgrade
$ sudo apt install cowsay
$ sudo apt remove cowsay
```

## Important Options

- `update`: Update package list
- `upgrade`: Upgrade software
- `install`: Install package
- `remove`: Remove package
