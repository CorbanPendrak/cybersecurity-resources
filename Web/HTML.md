[[The Web MOC]]
- - -

HTML stands for Hyper Text Markup Language. It formats data for the web using tags like `<html></html>`, but doesn't change based on user input.

Basic page:
```html
<html>
	<head>
		<title>Page title</title>
	</head>
	<body>
		<h1>Header</h1>
		<p>This is a paragraph...</p>
	</body>
</html>
```
